# Security

Authentication and compliance posture for Aurora.

## Single sign-on

Aurora supports single sign-on through SAML 2.0 identity providers such as
Okta and Azure AD. SAML is configured per organization by an admin.

## Compliance status

Aurora is not yet SOC 2 certified; a SOC 2 Type II audit is in progress. All
customer data is encrypted at rest with AES-256.
